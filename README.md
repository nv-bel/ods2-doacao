## Rodar o c√≥digo
1. Instale o PostgreSQL e crie o banco `ods_doacao`
2. Rode `npm install`
3. Crie uma venv python: `python3 -m venv venv` e ative a venv com `.venv\Scripts\activate`
4. Instale os requirements dentro da pasta backend `pip install -r backend/requirements.txt`

5. Com a venv ativa, inicie o backend com `python run.py`

6. Em outro terminal, inicie o frontend com `npm run dev`
